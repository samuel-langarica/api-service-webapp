Documentación de la API

URL Base

Todas las peticiones deben realizarse a la siguiente URL base:

https://api-as-a-service-template-zavwr.kinsta.app/api/v1

GET /status

Devuelve el estado básico del servidor.

Encabezados

Authorization: Bearer <tu_token>

Respuesta de Ejemplo

{
  "status": "ok"
}

Ejemplo con curl

curl -X GET "https://api-as-a-service-template-zavwr.kinsta.app/api/v1/status" \
  -H "Authorization: Bearer your_api_token"

GET /hello

Devuelve un mensaje sencillo de saludo.

Encabezados

Authorization: Bearer <tu_token>

Respuesta de Ejemplo

{
  "message": "Hello, World!"
}

Ejemplo con curl

curl -X GET "https://api-as-a-service-template-zavwr.kinsta.app/api/v1/hello" \
  -H "Authorization: Bearer your_api_token"
